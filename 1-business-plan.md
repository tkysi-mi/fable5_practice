# 日本のEdTech（社会人学習領域）新規事業リサーチ＆ビジネスプラン

- **作成日**: 2026-06-10
- **調査手法**: deep-researchワークフロー（5アングル並列Web調査 → 25ソース取得 → 120主張抽出 → 3票敵対的検証）＋一次ソース直接確認
- **前提条件**: 事業主体は既存企業の新規事業（ブランド・資金力あり）。VC型の赤字先行ではなく堅実な黒字化を目指すスモール〜ミドルビジネス志向。プロダクト形態はWeb/アプリに限定せず、物理教室・FC・OMOも検討対象。

**凡例**: ✅ = 多重検証済み or 一次ソースで直接確認 ／ 📄 = 一次ソースに引用ありだが多重検証は未了 ／ 💭 = 推定・仮定

---

## 結論（BLUF）

最有力案は **「中小企業特化型DXリスキリング研修事業（助成金活用伴走込み・OMO型）＋個人向け給付金指定講座への段階展開」** である。根拠は以下の4点。

1. 教育産業全体が横ばいの中で**BtoB研修・eラーニングだけが明確に成長**している
2. 助成金・給付金という**強力な制度レバーが2026年度末まで確実に存在**する
3. 矢野経済研究所自身が「リスキリング施策で確実に機能している事例は少ない」と指摘する**実行ギャップ＝アンダーサーブ**が存在する
4. アセットライトで「堅実な黒字化」という前提に最も合致する

## 1. 市場環境分析

### 1-1. マクロ: 全体は横ばい、社会人向けセグメントだけが成長

| 指標 | 数値 | 出典 |
|---|---|---|
| 教育産業全体（2024年度） | 2兆8,555.7億円（+0.7%）✅ | [矢野経研/日経](https://www.nikkei.com/article/DGXZRSP697748_X01C25A0000000/) |
| 同 2025年度予測 | 2兆8,720.6億円（+0.6%）✅ | 同上 |
| eラーニング市場（2024年度） | 3,812億円（+2.1%）✅ | [矢野経研](https://www.yano.co.jp/press-release/show/press_id/3795) |
| └ **BtoB（法人向け）** | **1,232億円（+7.8%）** ✅ | 同上 |
| └ BtoC（個人向け） | 2,580億円（**−0.4%**）✅ | 同上 |
| 企業向け研修サービス（2024年度） | 5,858億円（+4.6%）✅ | [矢野経研/日経](https://www.nikkei.com/article/DGXZRSP695138_X00C25A8000000/) |
| 同 2025年度予測 | 6,130億円（+4.6%）📄 | 同上 |

2024年度にプラス成長した7分野のうち4つ（資格・検定試験、語学スクール、企業向け研修、eラーニング）が社会人学習関連 ✅。少子化で子供向けが縮む中、**成長エンジンは社会人領域、その中でも法人経由（BtoB）に明確に偏っている**。

### 1-2. 警戒シグナル: BtoC直販と小規模教室は構造的に厳しい

- 個人向けeラーニングは微減（−0.4%）✅。通信教育業界はTSR調査で増収にもかかわらず合計18.8億円の赤字に転落、赤字企業率31.8%（競合激化・広告費高騰が原因）📄。**個人向けオンライン直販は広告費を燃やす消耗戦**になっている。
- 学習塾の倒産は2025年に55件と過去最多を更新 ✅、うち従業員5人未満が92.7% ✅（[東京商工リサーチ](https://www.tsr-net.co.jp/data/detail/1202296_1527.html)）。**単独小規模教室の自前運営はユニットエコノミクスが成立しにくくなっている**。
- 一方、FC型教室のユニットエコノミクス自体は健在。京進スクール・ワンのFC収支モデルは年商2,697万円・利益率37.9%・ロイヤリティ10%・初期投資220万円〜（物件費別）✅。つまり**「教室」という形態が死んだのではなく、集客力のない単独運営が死んでいる**、が正しい読み。

### 1-3. 制度環境: 2024〜25年に追い風が一気に強化された

| 制度 | 内容 | 確度 |
|---|---|---|
| 専門実践教育訓練給付金 | 受講費の**最大80%**支給（2024/10〜引き上げ）。内訳: 基本50%（年上限40万円）＋資格取得・就職で+20%（同16万円）＋賃金5%上昇で+10%（同8万円） | ✅ [厚労省PDF](https://www.mhlw.go.jp/content/001529622.pdf) |
| 民間EdTechの参入経路 | 「第四次産業革命スキル習得講座」（経産大臣認定、ITSSレベル3以上・30時間以上・2年以内）が給付指定の対象類型に含まれる | ✅ 同上 |
| **教育訓練休暇給付金（新設）** | **2025/10/1施行**。在職者が学習のため無給休暇を取ると基本手当相当を90/120/150日支給。長期・通学型の社会人学習を初めて制度が支える | ✅ [J-Net21](https://j-net21.smrj.go.jp/law/20250620.html) |
| 特定一般教育訓練給付金 | 資格取得で+10%追加支給を省令で新設 | ✅ 同上 |
| 人材開発支援助成金（人への投資促進コース・高度デジタル人材訓練） | 企業向け: 経費助成**75%（中小）/60%（大企業）**＋賃金助成**1,000円/時（中小）**（2025年度改正で960円から増額）。**令和8年度（2026年度）末までの時限措置** | ✅ [政府広報](https://www.gov-online.go.jp/article/202302/entry-9899.html)・[改正情報](https://www.internetacademy.co.jp/itlab/column-training/subsidy013.html) |
| 経産省「リスキリングを通じたキャリアアップ支援事業」 | **六次公募で新規受付終了 — 新規参入の窓口は閉じている** ✅。かつ補助事業者にはキャリア相談・講座提供・転職支援・フォローアップの4機能すべてが必須 ✅ | [公式](https://careerup.reskilling.go.jp/business/) |

戦略的含意は2つ:

1. **経産省ルート（リスキリング補助）は閉鎖済み → 使える制度レバーは「厚労省ルート」**（企業向け＝人材開発支援助成金、個人向け＝教育訓練給付金の講座指定）に集約される。
2. 矢野経済研究所は「リスキリング目的のサービスは増えたが、**現状において確実に機能している事例はそれほど多くはない**」と明言 ✅。供給は増えたが「修了して終わり」のコンテンツ提供型ばかりで、**成果（スキル定着・業務適用・賃金上昇）まで届く実行支援がアンダーサーブ**。

## 2. アンダーサーブ領域の特定

検証済みデータと公知の調査（パーソル・リクルートワークス等の社会人学習調査。日本の社会人の自己学習率が国際的に最低水準であること、ミドルシニア層のリスキリング支援が手薄であることは複数機関が一貫して指摘）を重ねると、需給ギャップは次の3つに集約される。

- **ギャップ①: 中小企業のDX人材育成** — 大手研修会社（リクルートMS、インソース等）の主戦場は大企業。中小企業は助成金（経費75%）を使えるのに、申請の複雑さと「何を学ばせればいいか分からない」で活用できていない。BtoB研修+7.8%成長の果実が中小に届いていない。
- **ギャップ②: 学習の「実行・定着」支援** — 矢野の指摘どおり ✅、コンテンツは過剰供給、伴走・成果担保は過少供給。
- **ギャップ③: ミドルシニア（40〜55歳）の在職リスキリング** — 教育訓練休暇給付金（2025/10施行 ✅）が長期学習を初めて可能にしたが、この制度を前提に設計された講座はまだほぼ存在しない（制度施行から1年未満の空白地帯）。

| 市場の追い風 | △ 制度は新しいがBtoC市場は微減 ✅ | ◎ BtoB研修+4.6〜7.8% ✅ | ○ 人手不足で構造需要 | × 学習塾倒産過去最多 ✅ |

## 4. 最有力案の詳細ビジネスプラン

### 「中小企業DXリスキリング・パートナー事業」（仮称）

#### 4-1. ターゲットと提供価値

- **一次ターゲット**: 従業員30〜300名の中小・中堅企業（製造・建設・卸小売・物流）。DXの必要性は認識しているが、研修の選定・助成金申請・受講後の定着まで自走できない層。
- **提供価値**: 「研修を売る」のではなく「**助成金診断 → スキル課題の特定 → 研修実施（集合＋eラーニングのOMO） → 業務適用までの伴走 → 助成金申請支援**」のワンストップ。矢野が指摘する「機能していないリスキリング」✅ の逆を張り、成果（業務での活用事例創出）までをパッケージに含める。
- **価格の魔法**: 1人30万円の研修でも、経費助成75% ✅ ＋賃金助成1,000円/時 ✅ で**中小企業の実質負担は1人あたり数万円**。営業上は「実質負担額」で提案でき、価格抵抗が極めて低い。

#### 4-2. 収益モデル

1. **研修パッケージ収入（主柱）**: 6ヶ月コース、1人30万円 💭、1社5〜10名単位
2. **助成金申請支援フィー**: 提携社労士経由、成果報酬の紹介料モデル 💭
3. **継続伴走サブスク**: 研修修了企業向け月額顧問（定着支援・次期人材の育成計画）💭
4. **（2年目以降）個人向け講座**: 第四次産業革命スキル習得講座の経産大臣認定を取得し、専門実践給付（最大80% ✅）対象の個人向け夜間・週末コースを展開 → **助成金時限切れ後の第2の柱**

#### 4-3. P/L試算

> 💭 仮定ベースの試算。検証済みなのは制度・市場データのみ。

| | Year 1 | Year 2 | Year 3 |
|---|---|---|---|
| 顧客社数 | 25社 | 60社 | 100社 |
| 受講者数（平均6名/社） | 150名 | 360名 | 600名 |
| 研修売上（@30万円） | 4,500万円 | 1.08億円 | 1.8億円 |
| サブスク・その他 | 300万円 | 1,500万円 | 4,000万円 |
| **売上合計** | **4,800万円** | **1.23億円** | **2.2億円** |
| 売上原価（講師業務委託・教材、〜40%） | 1,900万円 | 4,900万円 | 8,800万円 |
| 販管費（人件費4名→8名、マーケ） | 3,500万円 | 6,000万円 | 9,000万円 |
| **営業損益** | **▲600万円** | **+1,800万円** | **+4,200万円** |

- **初期投資**: 約2,000〜3,000万円（カリキュラム開発、LMS導入、初年度赤字補填）💭。物理教室は持たず、研修は顧客先・貸会議室で実施（学習塾倒産データ ✅ が示す固定費リスクを回避）。
- **損益分岐**: 18ヶ月前後 💭。既存企業の顧客基盤を流用できればYear 1から黒字も可。

#### 4-4. GTM戦略

1. **既存顧客基盤の転用（最重要）**: 事業主体の既存取引先への「助成金活用無料診断」をフックにしたクロスセル。CACをほぼゼロにできるのが既存企業発の最大の優位。
2. **紹介チャネルの構築**: 社労士事務所・地方銀行・商工会議所と提携（彼らは中小企業に助成金を勧めたいがコンテンツを持たない）。
3. **行政の認定獲得**: 人材開発支援助成金の対象訓練として実績を積み、2年目に第四次産業革命スキル習得講座認定 → 専門実践給付指定 ✅ を申請。
4. **パイロットで営業資産を作る**: 最初の10社は無償〜半額のパイロットで成果事例（Before/After）を作り、それを営業資産にする。

#### 4-5. リスクと対策

| リスク | 深刻度 | 対策 |
|---|---|---|
| **助成金の時限切れ**（人への投資促進コースは令和8年度末まで ✅） | 高 | (1) 2年目までに個人向け給付金指定講座（恒久制度）へ展開、(2) サブスク比率を高め助成金非依存の収益構造へ移行。教育訓練給付は雇用保険の恒久制度であり政策的にも拡充方向 ✅ |
| 助成金目当ての低質需要 | 中 | 経営者コミットメント（受講後の業務適用計画提出）を受注条件にする |
| 講師の確保・品質 | 中 | コア講師のみ雇用、専門講師は業務委託ネットワーク化。教材を標準化し講師依存度を下げる |
| 制度改正（助成率・要件変更） | 中 | 「実質負担額」前提の価格設計を毎年度見直し。社労士提携で改正情報を先取り |
| 大手研修会社の中小参入 | 低〜中 | 大手の営業構造は大口向けで小口多数は不採算。地域密着・伴走の深さで差別化 |

#### 4-6. 事業開始前の検証ステップ

事業開始前に検証すべき仮説は3つ。

1. 既存顧客の中小企業が「実質負担数万円」で6ヶ月研修にコミットするか（10社へのヒアリング＋パイロット2〜3社で検証）
2. 1人30万円の単価が助成金の経費助成限度額（100時間未満30万円/人 ✅）と整合するカリキュラム設計が組めるか
3. 修了者の「業務適用率」をKPIとして測定可能か

**この3つがパイロットで立証できなければA案（ミドルシニア向け）へピボット**、という撤退基準をあらかじめ置くことを推奨する。
