# なぜリスキリングは「需要があるのに機能しない」のか — 構造分析と対価設計

- **作成日**: 2026-06-10
- **位置づけ**: `1-business-plan.md` の補論。矢野経済研究所の「リスキリング目的のサービスは増えたが、確実に機能している事例はそれほど多くはない」という指摘（検証済み）の構造的原因を分析し、対価が成立するサービス設計を導く。

**凡例**: ✅ = 調査データで確認 ／ 💭 = 推論・仮説

---

## 結論（BLUF）

リスキリングが機能しない根本原因は、サービスの質の問題ではなく、**「学ぶ→使う→報われる」という因果連鎖が日本の雇用システムの中で3箇所すべて切断されている**ことにある。したがって「良いコンテンツを安く提供する」改善は的外れであり（むしろ価格競争を悪化させる）、対価が成立するのは**切断された連鎖を繋ぎ直す仕事**——すなわち (1) 業務との接続、(2) 時間の確保、(3) 処遇・市場価値への接続——に対してである。**サービスの本体は「教育」ではなく「組織への介入」であり、対価は研修予算ではなく経営課題解決の予算から取るべき**、というのが本稿の結論。

---

## 1. 問題の再定義: どこで連鎖が切れているか

リスキリングが価値を生むには、次の連鎖がすべて繋がる必要がある。

```
動機 → 時間 → 学習 → 転移（業務で使う） → 成果 → 報酬・処遇
```

日本の調査データは、この連鎖が**少なくとも3箇所で切断されている**ことを示している。

### 切断① 出口の切断: 学んでも報われない（→ 動機が生まれない）

- リクルート・Indeedの11カ国共同調査で、日本は「リスキリングへの意識」は4位と高いのに、**実践（学習時間）・キャリア自律・転職時に賃金が1割以上増えた人の割合・リスキリング効果の実感、のすべてで最下位** ✅（[リクルート](https://www.recruit.co.jp/blog/service/20241017_5230.html)）。
- 背景はメンバーシップ型雇用の構造: 職務記述書（ジョブ・ディスクリプション）の普及率が低く、賃金交渉の慣行もないため、**個人がスキルを獲得しても賃金・処遇に変換する経路が細い** ✅。
- 帰結として、自己啓発をしていない人は全体の約8割（正社員でも約6割）に上り、理由の最多は「やろうと思わない」✅（[マイナビ](https://career-research.mynavi.jp/column/20230516_48788/)）。
- 💭 これは個人の怠惰ではなく**合理的な無関心**である。報酬が期待できない投資をしないのは経済合理的な行動であり、「意識改革」を呼びかけるだけのサービスが機能しないのは当然の帰結。

### 切断② 時間の切断: 学ぶ時間が構造的に存在しない

- 帝国データバンク2024年調査: リスキリングに「取り組んでいる」企業はわずか8.9%。課題の最多は「**対応する時間が確保できない**」42.1%、次いで「対応できる人材がいない」38.9% ✅（[日経](https://www.nikkei.com/article/DGXZQOUC202OX0Q4A121C2000000/)）。
- 残業規制のパラドックス: 業務時間外の学習は残業規制・私生活との衝突で許容されず、業務時間内の学習は「遊んでいる」とみなされる——就業者が「時間的貧困」に陥っている ✅（能力開発基本調査の分析。なお教育訓練費を1円も支出していない企業が45.1% ✅）。
- 💭 eラーニングの「いつでもどこでも学べる」は、この文脈では**「いつまでも学ばなくてよい」**として機能する。時間を確保する強制力・正当化の仕組みがないコンテンツ提供は、構造的に消化されない。

### 切断③ 転移の切断: 学んでも業務で使われない

- 企業のリスキリング施策の失敗理由の最多は「**研修・学習内容が実務にマッチしていなかった**」、次いで「従業員任せになり成果に繋がらなかった」「対象者が学習を完了できなかった」✅（[マイナビニュース](https://news.mynavi.jp/article/20250704-3370058/)）。
- 国主導の職業訓練でも修了者の約3割が就職できておらず、訓練内容と採用ニーズのズレが指摘されている ✅（[日経](https://www.nikkei.com/article/DGXZQOUA143770U5A610C2000000/)）。
- 構造的要因 💭: 研修を主導する人事部門と、業務・事業変革を主導する事業部門の分断。「学ぶ場」と「使う場」が組織的に切り離されているため、学習内容を業務に持ち帰っても適用する権限・案件・上司の支持がない（研修転移 *transfer of training* の古典的問題）。

### 切断④（供給側）: 助成金が「実施」に払われ「成果」に払われない

- 💭 助成金・給付金は訓練の実施（時間数・修了）に対して支給され、成果（業務適用・賃金上昇）への連動はごく弱い（専門実践給付の賃金上昇+10%が例外的な始まり ✅）。このため供給側の合理的戦略は「**助成要件を満たす最低コストのコンテンツを量産する**」ことになる。
- さらに教育サービスは購入前に品質を判定できない経験財・信頼財であり、買い手が質を見分けられない市場では価格だけが比較軸になる（レモン市場化）💭。これは通信教育業界が増収でも赤字転落（赤字企業率31.8%）した事実 📄 と整合する。
- 帰結: **「需要はあるのに機能している事例が少ない」のは、機能しないサービスが市場から淘汰されない（助成金で延命される）ため** 💭。矢野の指摘はこの均衡の観察と読める。

## 2. 含意: 何に対価が成立し、何に成立しないか

| 提供物 | 対価の成立 | 理由 |
|---|---|---|
| コンテンツ・講座そのもの | ✗ 構造的に縮小 | 切断④のレモン市場化＋無料・低価格教材（動画、生成AI）との競争。質を上げても買い手が判別できない |
| 「受講させた」という実績 | △ 助成金がある間のみ | 制度時限（2026年度末）とともに消える需要 |
| **時間と強制力の設計**（学習が起きる状態を作る） | ○ | 切断②を繋ぐ。個人の意思に頼らない仕組みは組織にしか作れず、外部の介入に希少性がある |
| **業務への転移**（実務課題で学び、成果物が業務改善） | ◎ | 切断③を繋ぐ。成果が可視化されればROIで語れる＝研修予算ではなく事業予算から払える |
| **処遇・市場価値への接続**（資格・認定・人事制度反映） | ○ | 切断①を繋ぐ。個人の学習動機を初めて起動する。「会社から推奨資格を明示されれば8割が取り組む」✅ |

ここから導かれる設計原理: **教育サービスの粗利は「教えること」ではなく「連鎖の切断を繋ぐこと」から生まれる。** そして繋ぐ作業は本質的に組織コンサルティングであり、コンテンツより複製困難＝価格競争に巻き込まれにくい 💭。

## 3. サービス設計: 「育成型業務改善プログラム」

前レポートのB案（中小企業DXリスキリング）を、上記の原因分析に合わせて再設計する。

### 3-1. 設計原則（原因 → 打ち手の対応）

| 切断 | 打ち手 | 具体策 |
|---|---|---|
| ③ 転移 | **実業務課題を教材にする**（アクションラーニング） | 汎用カリキュラムではなく、顧客企業の実課題（例: 受発注業務の自動化、データ集計の脱属人化）をプロジェクト題材に設定。**研修の成果物＝実際の業務改善**。学ぶ場と使う場を物理的に一致させる |
| ② 時間 | **就業時間内に埋め込む** | 経営者との契約で週N時間の業務時間内学習を確保（賃金助成1,000円/時 ✅ が会社側の逸失コストを直接補填——制度がこの打ち手のためにあると言ってよい）。隔週の対面セッション（OMO）で締切と進捗の強制力を作る |
| ① 報酬 | **処遇への接続を契約に含める** | 修了者のスキル認定（給付金指定資格と連動）＋顧客企業の人事評価項目への反映を経営者と事前合意。「学んだら何が変わるか」を受講開始前に本人へ明示 |
| ④ 質の不可視 | **成果KPIの事前合意と公開** | 「業務適用率」「削減工数」「改善事例数」を着手前に定義し、修了時にレポート化。営業資産にもなる |
| 全体 | **受講者ではなく経営者を顧客にする** | 失敗最多原因「従業員任せ」✅ の逆。経営者のコミットメント（時間確保・適用権限・処遇反映）を受注条件とし、満たさない案件は受けない |

### 3-2. 商品構成と対価設計

対価設計の核心は「**どの予算から支払われるか**」の選択にある 💭:

- 研修予算: 小さく、価格競争的。1人数万円の世界 → 避ける
- **業務改善・DX予算: 大きく、ROIで正当化される。1プロジェクト数百万円の世界 → ここを取る**
- 助成金: 強力だが時限 → 価格の「実質負担引き下げ装置」としてのみ使い、収益構造の前提にしない
- 個人＋給付金: 恒久制度 → 第2フェーズの柱

| 商品 | 内容 | 価格 💭 | 支払い元 |
|---|---|---|---|
| ① 診断・設計（入口） | 業務課題の棚卸し＋育成計画＋助成金設計（2〜4週間） | 30〜50万円/社 | DX予算（助成金診断を無償フックに） |
| ② 育成型業務改善プログラム（主柱） | 6ヶ月。受講者5〜8名が実業務課題を題材にDXスキルを習得し、改善を実装。隔週対面＋eラーニング＋メンター伴走 | 250〜400万円/社（=1人40〜50万円相当）。経費助成75% ✅＋賃金助成適用で実質負担はおよそ3分の1〜4分の1 | DX予算＋助成金 |
| ③ 成果マイルストーン報酬 | 事前合意KPI（業務適用2件以上、削減工数など）達成時 | 基本フィーの10〜20%を上乗せ | 同上 |
| ④ 継続伴走サブスク | 修了後の定着支援・社内展開・翌期の育成計画 | 月10〜20万円/社 | 経営企画予算 |
| ⑤ 個人向け給付指定講座（2年目〜） | ②のカリキュラムを汎用化し経産大臣認定（第四次産業革命スキル習得講座 ✅）→ 専門実践給付（最大80% ✅）対象の夜間・週末コース | 30〜50万円/人（実質負担6〜10万円） | 個人＋雇用保険（恒久制度） |

成果報酬の設計上の注意 💭: **完全成果報酬は推奨しない**。成果の帰属（研修のおかげか、他要因か）の測定コストが高く、紛争リスクと収益の不安定化を招く。「基本フィーで原価を回収し、検証可能な軽量マイルストーン（適用事例数など客観カウント可能なもの）で上乗せ」のハイブリッドが、買い手への品質シグナリングと事業の堅実性を両立する。

### 3-3. この設計が価格競争を回避できる理由

1. **比較対象が変わる**: 「1人5万円のeラーニング」とではなく「数百万円の業務改善コンサル」と比較される。後者の相場では②の価格はむしろ安い 💭
2. **成果が残る**: 修了時に業務改善という検証可能な成果物が残るため、経験財の不可視性（切断④）を部分的に解消し、紹介・継続につながる
3. **複製困難**: コンテンツはコピーできるが、経営者合意の取り付け・業務課題の題材化・伴走は属人的な営業/コンサル能力であり、量産型事業者が模倣しにくい 💭

### 3-4. この設計のコストとリスク（正直な留保）

- **スケールしにくい**: 顧客ごとのカスタマイズと伴走は労働集約的。粗利率はコンテンツ型より低く、成長は人材採用に律速される 💭。→ 題材のパターン化（業種別テンプレート）と⑤の汎用講座で限界費用を下げるのが2年目以降の課題
- **受注の選別が必要**: 経営者コミットメントを条件にすると失注が増える。ただしこれは品質の担保コストであり、「従業員任せ」案件を受けて失敗事例を作るほうが長期的に高くつく 💭
- **成果KPIの設計力が競争力の核**: 測定が雑だと成果報酬が紛争の種になる。初期パイロットでKPI定義の型を作ることが最重要の学習目標

---

## 4. まとめ

「機能している事例が少ない」のは市場の未成熟ではなく、**助成金駆動のコンテンツ供給が、切断された因果連鎖の上に乗っているから**である。逆に言えば、連鎖を繋ぐ設計——実業務題材・時間の契約的確保・処遇接続・成果の可視化——を商品の中核に置く事業者にとって、この「機能していない」状態こそが参入機会となる。対価は「教えること」にではなく「成果が出る状態を組織に作ること」に課金する。これが、価格競争と助成金時限の両方を生き延びる唯一の経路だと考える 💭。
